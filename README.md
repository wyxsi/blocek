# 🧾 Rozdeľovač bločkov

Aplikácia na rozdelenie nákupu z bločku medzi 2 ľudí.

## Použitie

1. Stiahni `index.html` a otvor v prehliadači
2. Exportuj PDF bločku z [ebločkov.sk](https://www.eblkov.sk)
3. Nahraj PDF bločku
4. Položky sa automaticky extrahujú
5. Uprav / pridaj položky podľa potreby
6. Prirad každú položku:
   - 👤 Osoba 1
   - 👤 Osoba 2
   - ½ Oba (rozdeliť na polovicu)
   - ❌ (vylúčiť)
7. Zobrazia sa 2 sumy

## Funkcie

- 📄 PDF extrakcia položiek
- ✏️ Úprava názvov a cien
- ➕ Pridanie / vymazanie položiek
- ✓ Porovnanie súčtu s PDF
- 🏪 Podpora: Kaufland, Tesco, Lidl, Action

## Poznámka

PDF bločky exportuj z portálu [ebločkov.sk](https://www.eblokov.sk) - oficiálny register elektronických bločkov.
